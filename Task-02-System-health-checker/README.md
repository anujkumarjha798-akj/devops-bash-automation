# System Health Checker

A Bash script that monitors the health of a Linux system by checking:

- Disk Usage
- RAM Usage
- CPU Usage
- Overall System Health
- Report Logging

## Features

- Checks Disk Usage
- Checks RAM Usage
- Checks CPU Usage
- Displays Overall Health Status
- Saves reports to a log file

## Requirements

- Linux
- Bash
- Root Privileges

## Usage

```bash
chmod +x system-health-check.sh
sudo ./system-health-check.sh
